= Checkout API
:toc: left
:toclevels: 2

== Overview

The Checkout API calculates the total price of a cart, applying any active promotional offers.

Base URL: `http://localhost:8080`

== Endpoints

=== POST /api/v1/checkout

Calculates the price for a cart. Applies active offers where eligible.
